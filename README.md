# TravelWithMe

A minimal single-page travel blog built with AngularJS.


## Developement
```bash
git clone https://github.com/SureshAmal/travel-with-me.git
cd travel-with-me
```


## Run Locally

**Using live-server** (auto-reloads on changes, with SPA fallback):

```bash
npx live-server --entry-file=index.html
```

**Using Python:**

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`

## Tech

- AngularJS 1.8
- Vanilla CSS (dark/light theme)
